# TrueNAS 存储相关 Shell 脚本

硬盘状态监控、空间检测、备份等脚本。

## 脚本列表

| 脚本 | 说明 |
|------|------|
| `备份执行脚本.sh` | 7z 压缩备份，配合 `备份配置.conf` 使用 |
| `检测网盘剩余空间.sh` | 定期检测 OneDrive/云盘剩余空间并推送通知 |
| `检测硬盘剩余空间.sh` | 检测本地硬盘剩余空间并推送通知 |
| `检测硬盘状态.sh` | 使用 smartctl 检测硬盘健康状态 |
| `硬盘休眠脚本.sh` | 管理硬盘休眠策略 |
| `硬盘休眠与状态监控.sh` | 硬盘休眠 + 状态综合监控 |
| `重要目录备份.sh` | 备份指定重要目录 |
