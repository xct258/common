# 自动更新 GitHub 文件

检查 GitHub 仓库的指定文件是否有更新，自动覆盖本地文件。

## 脚本列表

| 脚本 | 说明 |
|------|------|
| `更新github文件.sh` | 核心脚本：检查 GitHub 文件 SHA 与本地 MD5，有更新则下载覆盖 |
| `自动更新github文件调用.sh` | 调用脚本：配置需要更新的文件列表并循环调用 |

## 核心脚本 `更新github文件.sh`

支持参数：

```bash
bash 更新github文件.sh <本地路径> <仓库名> <文件路径> [Token] [分支]
```

- `本地路径`: 本地文件路径
- `仓库名`: GitHub 仓库名（如 `user/repo`）
- `文件路径`: 文件在仓库中的路径
- `Token`: GitHub Token（可选，私有仓库需要）
- `分支`: 分支名（默认 `main`）

## 返回值

| 退出码 | 说明 |
|--------|------|
| 0 | 成功完成（无论是否有更新） |
| 2 | 参数错误 |
| 3 | 网络或组件错误 |

## 输出 JSON

每次执行输出标准 JSON：

```json
{"status":"ok","message":"...","updated":true}
```
