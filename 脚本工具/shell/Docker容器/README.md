# Docker 相关 Shell 脚本

批量管理 Docker 镜像的工具脚本。

## 脚本列表

| 脚本 | 说明 |
|------|------|
| `批量获取docker镜像.sh` | 从配置的镜像列表中批量拉取、打包为 tar 文件 |
| `批量释放docker镜像.sh` | 从指定目录批量导入 tar 镜像文件到 Docker |

## 批量获取使用

在 `image_map` 数组中配置需要拉取的镜像和自定义打包名称，运行后自动：

1. 拉取镜像
2. 保存为 `.tar` 文件到 `./docker镜像/<日期>/`
3. 删除本地镜像（释放空间）

## 批量释放使用

将 `.tar` 镜像文件放入指定目录，运行脚本自动逐个执行 `docker load -i`。
