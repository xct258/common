# Flare 书签导航页

自部署的浏览器起始页与书签导航，支持自定义应用快捷方式。

## 部署信息

- **镜像**: `soulteary/flare:latest`
- **端口**: 127.0.0.1:5005（仅本地监听）
- **数据卷**: `./app:/app`

## 环境变量

| 变量 | 值 | 说明 |
|------|-----|------|
| `FLARE_DISABLE_LOGIN` | 0 | 启用登录认证 |
| `FLARE_USER` | xct258 | 登录用户名 |
| `FLARE_PASS` | (已配置) | 登录密码 |

## 访问

通过 Nginx 反向代理暴露到公网访问，应用数据保存在 `./app` 目录中，包括书签和应用配置。
