# Syncthing 文件同步

开源 P2P 文件同步工具，在多个设备之间实时同步文件夹。

## 部署信息

- **镜像**: `syncthing/syncthing:latest`
- **主机名**: oci-1-debian-1-syncthing
- **时区**: Asia/Shanghai
- **运行用户**: root

## 端口

| 端口 | 协议 | 说明 |
|------|------|------|
| 127.0.0.1:8384 | TCP | Web 管理界面（仅本地） |
| 22000 | TCP/UDP | 设备间同步端口 |
| 21027 | UDP | 局域网发现协议 |

## 数据卷

| 宿主机路径 | 容器路径 | 说明 |
|-----------|---------|------|
| `./data` | `/var/syncthing` | Syncthing 配置与数据 |
| `/` | `/data1` | 宿主机根目录挂载（用于同步任意路径） |

## 使用

Web 管理界面仅本地监听，通过 SSH 隧道或反向代理访问。首次启动后需配置设备 ID 和共享文件夹。
