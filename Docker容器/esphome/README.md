# ESPHome 智能设备管理

开源 IoT 固件管理系统，支持通过 YAML 配置生成 ESP8266/ESP32 固件并 OTA 更新。

## 部署信息

- **镜像**: `esphome/esphome`
- **Web 端口**: 6052（host 网络模式）
- **配置目录**: `./data:/config`

## 配置文件示例

`配置文件示例/` 目录包含以下设备的 YAML 配置：

| 文件 | 说明 |
|------|------|
| `esp01s-1` | ESP-01S 继电器开关配置 |
| `esp32-1` | ESP32 基础配置 |
| `esp8266-1` | ESP8266 基础配置 |
| `VL53L1X配置文件` | VL53L1X 激光测距传感器配置 |
| `开关控制esp实体` | 开关控制实体定义 |
| `开关配置` | 通用开关配置模板 |

## 使用

1. 启动容器后访问 `http://<IP>:6052`
2. 通过 WebUI 编辑配置或上传现有 YAML 文件
3. 编译并连接 ESP 设备刷入固件
