# Vaultwarden 密码管理器

自托管的密码管理器，与 Bitwarden 客户端完全兼容。

## 部署信息

- **镜像**: `vaultwarden/server:latest`
- **端口**: 127.0.0.1:8808（仅本地监听）
- **数据卷**: `./data/:/data/`

## 安全配置

| 环境变量 | 值 | 说明 |
|---------|-----|------|
| `SIGNUPS_ALLOWED` | false | 关闭公开注册，仅管理员添加用户 |

## 访问

通过 Nginx 反代对外提供 HTTPS 访问，支持所有 Bitwarden 官方客户端（浏览器、手机、桌面）连接。

## 功能

- 密码、笔记、身份信息、支付卡安全存储
- 端到端加密
- 自动填充
- 组织共享
- 两步验证
