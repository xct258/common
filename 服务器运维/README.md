# 服务器运维

服务器运维相关的配置、脚本和文档，涵盖虚拟化、网络、存储、监控等。

## 根目录文件

| 文件 | 说明 |
|------|------|
| `甲骨文云添加ipv6和多ip.txt` | Oracle Cloud 实例添加 IPv6 和多个辅助 IP 的网络配置方法 |

## 子目录

| 目录 | 说明 |
|------|------|
| `LXD容器/` | LXD 系统容器创建与管理命令（Debian 13、Docker 嵌套等） |
| `OpenMediaVault/` | OMV NAS 系统安装与插件配置（硬盘加密、iSCSI、Docker） |
| `OpenWrt/` | OpenWrt 软路由配置（PVE 导入、OpenClash、Zerotier、主路由设置） |
| `other/` | 其他系统配置（rc.local 开机自启） |
| `PVE/` | Proxmox VE 虚拟化配置（硬件直通、内存挂载、软件源） |
| `Telegram通知/` | Telegram Bot API Token 记录（多级别通知） |
| `rclone/` | rclone 云盘挂载配置（OneDrive 私有 API、Cloudflare R2） |
| `服务监控/` | PHP 服务监控面板（读取 INI 状态文件展示服务状态） |
| `服务器重装脚本/` | 服务器系统一键重装脚本（bin456789/reinstall） |
